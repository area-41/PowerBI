# 📊 Formação Power BI - Microsoft & Santander Academy

Este diretório contém o progresso técnico e os projetos práticos desenvolvidos durante o curso de Power BI oferecido pela **Microsoft** em parceria com a **Santander Academy**. O foco desta etapa foi o domínio da ferramenta desde a ingestão de dados até a criação de relatórios interativos e limpeza de dados com Power Query.

## Competências Desenvolvidas

### 1. Ingestão e Estruturação de Dados
*   **Conectividade:** Importação de dados a partir de fontes variadas (Excel, Tabelas Estruturadas, arquivos CSV e TXT).
*   **Tratamento de Dados (Power Query):** Primeiros passos com o editor de consultas, alteração de fontes de dados, transformações básicas e carga otimizada para o modelo.
*   **Tipagem e Formatação:** Aplicação de formatos padrão e personalizados para garantir a integridade da análise.

### 2. Visualização de Dados e Storytelling
*   **Construção de Dashboards:** Criação de relatórios do zero utilizando melhores práticas de layout e Window Management.
*   **Objetos Visuais:** Domínio de Cartões (KPIs), Gráficos de Barras, Linhas e Combinações para análise de tendências.
*   **Interatividade:** Configuração de interações entre visuais e propriedades de formatação para melhorar a experiência do usuário.

### 3. Análise e Filtros Avançados
*   **Painel de Filtros:** Implementação de filtros básicos, filtros em campos totalizados e utilização de **Top N** para destaques de performance.
*   **Formatação Condicional:** Uso estratégico de Barras de Dados e Ícones para facilitar a leitura visual de métricas críticas.

## Ferramentas Utilizadas
*   **Power BI Desktop**
*   **Power Query** (M Language para transformações)
*   **Microsoft Excel** (Fontes de dados)
