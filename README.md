# Power BI & Data Visualization
Dashboards, reports e estudos usando o Power BI da Microsoft.

Este repositório centraliza meus projetos, estudos e implementações de Business Intelligence utilizando **Power BI**. O objetivo é demonstrar a aplicação de técnicas de ETL, modelagem de dados (DAX) e Storytelling para transformar dados brutos em decisões estratégicas.

> **Status:** Em desenvolvimento constante 🚀

---

### Organização do Repositório

O repositório está estruturado para separar o aprendizado acadêmico de aplicações de mercado e projetos autorais:

*   **[./estudos-microsoft-santander](./estudos-microsoft-santander):** Projetos e laboratórios práticos finalizados através da formação Microsoft/Santander Academy. Foco em fundamentos de Power Query, visualização básica e filtros avançados.
*   **[./dashboards-profissionais](./dashboards-profissionais):** (Em breve) Dashboards focados em métricas de negócio, utilizando fontes de dados complexas e modelos estelares (Star Schema).
*   **[./integracoes-python-ml](./integracoes-python-ml):** (Em breve) Experimentos integrando scripts Python dentro do Power BI para análises preditivas e tratamento avançado de dados.


### Stack Tecnológica & Skills

Neste repositório, aplico as seguintes competências:

*   **ETL & Limpeza de Dados:** Uso intensivo de **Power Query (M Language)** para normalização e automação de cargas.
*   **Modelagem de Dados:** Criação de relacionamentos, hierarquias e medidas calculadas com **DAX**.
*   **Storytelling:** Design de dashboards focado na experiência do usuário (UX) e clareza de KPIs.
*   **Integração:** Conectividade com diversas fontes, incluindo Excel, CSV e bancos de dados SQL.

### Pasta: Formação Microsoft & Santander Academy

Trilha de especialização técnica da Microsoft com conhecimentos em:
1.  **Ingestão de Dados:** Importação estruturada e formatação de colunas.
2.  **Relatórios Interativos:** Uso de Cartões, Gráficos de Combinação e formatação de propriedades.
3.  **Análise Avançada:** Filtros de Top N e Formatação Condicional baseada em ícones e barras.
4.  **Power Query:** Operações básicas e transformações de fontes dinâmicas.
